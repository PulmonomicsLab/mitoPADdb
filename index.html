mitoPADdb - Mitochondrial Proteins Associated with Diseases database

The mitoPADdb is a manually curated database that provides comprehensive information on 813 mitochondrial protein-encoding (MPE) genes associated with 1793 disease terms. It contains information on mutations and expressional variations of MPE genes at both the mRNA and protein levels. The database classifies the diseases linked with mitochondrial proteins into 14 broad categories. It allows users to access the data using (I) by search option with Gene symbol or UniProt accession and (II) by browse options. The search option provides information about 18356 mutations, 1286 expressional variations (mRNA and protein levels) with experimental samples and techniques, and associated diseases with source identifiers.
